安装Hugo（Windows）

  1. 在GitHub上下载压缩包（最慢的方法）(==因为不支持SCSS,需要安装拓展版本==)
  2. 在E盘创建一个名为hugo的文件，在hugo文件下创建一个bin文件
  3. 将下载的压缩包解压到bin文件中
  4. 配置环境变量，在系统变量path中添加bin文件的路径
  5. 使用cmd或者powershell，输入hugo help和hugo version ,查看是否安装成功
  6. 在E盘中用powershell 输入hugo new site myblog,创建一个个人博客。
  7. 安装Git
  8. 在hugo的官网中用Git拉取一个主题（==powershell管理员模式==）
  9. 部署到远端Github上（GitHub需要与git绑定）
  10. cmd也可以启动hugo，不需要是用管理员模式（第四次启动尝试)
  11. 终端需要管理员模式，因为使用git需要管理员模式才行

Hugo的终端指令

  1. hugo new post/blog01.md （新建一篇文章）

  2. hugo server –theme=m10c –buildDrafts （本地启动项目）

  3. 将个人博客部署的到远端github

    • hugo –theme=m10c –baseURL=“https://czxcw.github.io” –buildDraft（会在根目录下生成一个public文件）
    • git init （让public文件成为本地仓库，初始化一个git仓库）
    • git remote add origin https://github.com/czxcw/czxcw.github.io.git(第一次才使用此命令，让本地仓库和github仓库关联)
    • git add .(将所有文件都添加到暂存区)[在public文件下使用]
    • git commit -m “提交信息”（将暂存区的文件提交到当前的分支）
    • git push -u origin master（将本地仓库推到远端的GitHub仓库上）

如何更新博客和发布新的博客

  1. hugo new post/blog01.md （新建一篇文章）{如果要发布新的博客，使用此命令}
  2. hugo server –theme=m10c –buildDrafts （本地启动项目）
  3. hugo –theme=m10c –baseURL=“https://czxcw.github.io” –buildDrafts
  4. git add .(将所有文件都添加到暂存区)[在public文件下使用]
  5. git commit -m “提交信息”（将暂存区的文件提交到当前的分支）
  6. git push -u origin master（将本地仓库推到远端的GitHub仓库上）
